Privacy Policy — Captain's Toonz Tools

Last updated: April 2026

Captain's Toonz Tools is a Chrome extension built for the Degen Toonz NFT community. This privacy policy explains how the extension handles data.

Data We Do Not Collect

Captain's Toonz Tools does not collect, store, transmit, or share any personal information. Specifically, we do not collect:

  • Names, email addresses, or any personally identifiable information
  • Browsing history or web activity
  • Financial or payment information
  • Location data
  • Authentication credentials

Data Stored Locally

The extension stores the following data locally on your device using Chrome's built-in storage API. This data never leaves your device:

  • Detonation Index — a locally built map of Detonated Toonz token IDs used for instant lookups
  • Tab state — the last viewed tab and results, so the extension restores your session when reopened

External API Calls

The extension makes requests to the following public APIs to fetch NFT data:

  • Alchemy (alchemy.com) — for NFT metadata and on-chain claim status
  • OpenSea (opensea.io) — for collection statistics and listing prices
  • Ethereum blockchain — for direct on-chain contract calls

These requests contain only NFT token IDs and wallet addresses that you explicitly enter. No personal data is transmitted.

Wallet Addresses

If you use the Wallet Toonz Check feature, the wallet address you enter is sent to Alchemy solely to retrieve the list of NFTs owned by that address. This address is not stored, logged, or shared.
